## How to Use

1. Open `index.html` in your web browser
2. Click "Get Started - It's Free!" to enter the app
3. Browse failures in the feed
4. Click the "+" button to share your own failure
5. Visit your profile to see stats and badges

## Features Implemented

### Landing Page
- Eye-catching hero section with gradient background
- Fun, engaging copy ("Join and connect with over 1,000 failures like you!")
- Feature highlights with emoji icons
- Testimonials from "users"
- Smooth animations and transitions

### Feed Page
- Sample failure posts with user avatars
- Interactive engagement buttons (like, comment, share)
- Tags for categorization
- Bottom navigation bar
- Smooth animations on load

### Create Post Page
- Emoji picker for emotional expression
- Text area with character counter (500 max)
- Tag system (up to 5 tags)
- Optional lesson learned field
- Form validation
- Success notifications

### Profile Page
- Profile header with avatar and bio
- Stats display (Failures, Supporters, Lessons)
- Achievement badges system (locked/unlocked)
- Recent failures timeline
- Interactive elements with hover effects

## Interactive Features

- **Like System**: Click heart icon to like/unlike posts
- **Emoji Selection**: Choose how you feel about your failure
- **Tag Management**: Add/remove tags dynamically
- **Character Counter**: Real-time feedback on post length
- **Animations**: Smooth transitions and engaging interactions
- **Notifications**: User feedback for various actions
